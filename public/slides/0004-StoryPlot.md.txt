# how ES Lint may help

```javascript className=codemedium
// Restrict eslint rules for Binary Operations
rules: {
	'no-restricted-syntax': ['error', {
		selector: 'BinaryExpression[operator=\'+\']',
		message : 'syntax protocol error: +'
	}, {
		selector: 'BinaryExpression[operator=\'>\']',
		message : 'syntax protocol error: >'
	}],
}
```
